// 輸入 numCourses = 4 prerequisites = [[0,1],[1,2],[1,3]] // 邊 0→1, 1→2, 1→3 queries = [[0,3],[2,3],[0,2]] indeg = [0,1,1,1] queue=[0] 圖: 0 → 1 →(2, 3)
// 拓撲序處理 + 傳遞 處理 0:table[0][1]=T (0 沒先修,不傳遞) push 1 處理 1:table[1][2]=T;傳遞 0 → table[0][2]=T push 2 table[1][3]=T;傳遞 0 → table[0][3]=T push 3 處理 2:無出邊 處理 3:無出邊 最終 table: 0→{1,2,3} 1→{2,3}
// 查詢(直接查表) [0,3] → table[0][3] = true (0→1→3) [2,3] → table[2][3] = false (2 到不了 3) [0,2] → table[0][2] = true (0→1→2)
注意 [2,3] 是 false —— 2 和 3 是兄弟(都由 1 指出),彼此之間沒有先修關係。 傳遞只沿著箭頭方向往下走,不會在同層之間橫向連結。這正是「可達性」和「同一個父節點」的區別。

應用題// APPLY

App 1. 若題目不保證無環(可能有環),這套拓撲序傳遞會怎樣?要先做什麼?
App 2. 若 V 很大(上萬)但邊很稀疏,O(V²) 的 table 記憶體吃不消,怎麼辦?(提示:bitset / 每點存可達集合)
App 3. 用 Floyd-Warshall 重寫這題,三重迴圈長什麼樣?為什麼它不需要先拓撲排序?
// ANSWERS
A1. 傳遞的正確性靠一個前提:處理 u 時,「誰是 u 的先修」(即 table[i][u] 這一欄)必須已經算完。拓撲序恰好保證這件事 —— u 出隊時所有前驅都已先出隊、已把自己寫進 table[*][u]。若亂序處理,可能在 u 的某個前驅還沒處理時就先處理 u,那條前驅關係還沒寫進 table[*][u],傳給後繼時就漏掉了,結果不完整。
A2. 「傳遞」就是把「u 的先修」全部變成「v 的先修」——因為 u→v,任何能到 u 的,經過 u 就能到 v。table[i][u] 的意思是「i 是 u 的先修」(i 能到 u),這正是我們要往下遞給 v 的;所以複製 table[i][u] → table[i][v]。而 table[u][i] 是「u 是 i 的先修」,方向相反、不是我們要的。行固定看「誰指向這一欄的節點」。
A3. 差在誰指向誰。本題 [a,b] 明確是「a 是 b 的先修」,所以邊 a→b、indeg[b]++。而 207/210 的 [a,b] 是「要上 a 得先上 b」,先修是 b,邊要建成 b→a。判斷法:找出「誰要先上」,箭頭就從先上的那門指向後上的那門;或看題目給的入度該加在誰身上。讀錯方向整份答案會顛倒,務必先確認。
A4. 外層拓撲把每條邊 u→v 走一次,共 E 條。每走一條邊,內層 for i 掃過所有 V 個可能的前驅做傳遞 → 每條邊 O(V)。合起來傳遞部分是 O(V·E),加上拓撲骨架 O(V+E),總共 O(V·(V+E))。內層那個 for i 正是多出來的 V 因子。
A5. 當 V 小(這題 ≤ 100)時,Floyd-Warshall 三行搞定、不用建圖也不用拓撲,寫起來更快更不易錯,O(V³) 在 V≤100 完全能接受。取捨點:V 小、圖偏稠密、想少寫程式 → Floyd;V 較大、圖稀疏、想壓時間 → 拓撲序傳遞(O(V·(V+E)) 在稀疏圖優於 V³)。
App 1. 有環時 Kahn 剝不完:環上的節點入度永遠不歸零、進不了佇列,它們的 table 欄位就不會被完整填。所以要嘛題目保證 DAG,要嘛先判環(見 207);若真有環,可達性定義本身也要重新界定(環內互相可達)。本題保證無環,所以直接傳遞。
App 2. 把每個節點的「先修集合」用 bitset<V> 存,傳遞時 reach[v] |= reach[u] 一次做完一整列的位元或,常數大幅下降、記憶體是 O(V²/64)。若圖非常大且稀疏、可達集合也小,可改存 vector/unordered_set 只放真正可達的點,避免 V² 的密集矩陣。
App 3. 先把直接邊寫進 reach,再三重迴圈:for k: for i: for j: reach[i][j] = reach[i][j] || (reach[i][k] && reach[k][j]);。含義是「若 i 能到 k、k 能到 j,則 i 能到 j」,用每個 k 當中繼點逐步補全。它不需要拓撲序,因為它枚舉所有可能的中繼點、把所有間接路徑都考慮到了 —— 順序無關,最外層跑完每個 k 就收斂。代價是固定 O(V³)。
