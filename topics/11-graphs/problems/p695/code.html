完整實作 · DFS 量面積// SINK + COUNT

p695.cpp C++17 
class Solution {
private:
    const vector<pair<int, int>> dirs = {{-1, 0}, {1, 0}, {0, -1}, {0, 1}};
    int m, n;

public:
    int maxAreaOfIsland(vector<vector<int>>& grid) {
        m = grid.size(); n = grid[0].size();
        int maxArea = 0;

        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                if (grid[i][j] == 0) continue;       // 水 / 已沉,跳過(int:不帶引號)

                int curArea = 0;
                dfs(grid, i, j, curArea);          // 沉整座島並數面積
                maxArea = max(maxArea, curArea);      // 記最大的一座
            }
        }
        return maxArea;
    }

    void dfs(vector<vector<int>>& grid, int r, int c, int& curArea) {
        if (r < 0 || r >= m || c < 0 || c >= n || grid[r][c] == 0) return;
        grid[r][c] = 0;                        // 立刻沉掉(標記已訪)
        curArea++;                              // 這一格算面積 +1
        for (auto [dr, dc] : dirs) {
            int nr = r + dr, nc = c + dc;
            dfs(grid, nr, nc, curArea);
        }
    }
};

複雜度分析// COMPLEXITY

T = O(m·n)　　S = O(m·n)

每格被掃一次、被 DFS 沉一次(順便 curArea++)。總造訪量 O(mn)。空間為遞迴堆疊深度,最壞可達 O(mn)。

步驟成本說明
雙迴圈掃描O(mn)每格看一次
DFS 沉島 + 計面積O(mn)每格最多沉一次、+1 一次
遞迴堆疊O(mn)最壞深度

注意 · 邊界與陷阱// CAUTION · PITFALLS

陷阱後果正解
誤加引號 == '0'型別錯(這題是 int)grid 是 int → == 0、= 0 不帶引號
curArea 傳值而非參考面積回不到上層,算錯用 int& curArea,或 dfs 回傳 1+Σ
遞迴鄰居前沒先沉互遞迴 + 重複計數進 dfs 先 grid=0; curArea++
maxArea 初值設錯全水時答案錯初值 0,沒島自然回 0
base case 少了出界檢查越界存取 · UB先驗界再讀 grid[r][c](|| 短路)
