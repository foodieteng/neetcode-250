寫法 A · DP(你的版本)// CUT vs NO-CUT

p343.cpp C++17 
class Solution {
public:
    int integerBreak(int n) {
        vector<int> dp(n + 1, 1);          // dp[i] = i 拆成 ≥2 塊的最大乘積

        for (int i = 2; i <= n; i++) {
            for (int j = 1; j < i; j++) {      // 切點 j:分成 j 與 i-j
                dp[i] = max(dp[i], max(j * dp[i - j],   // 切:i-j 繼續拆
                                        j * (i - j)));  // 不切:i-j 留整塊
            }
        }

        return dp[n];
    }
};

那兩項就是「切 / 不切 i−j」:j*dp[i−j] = 把 i−j 繼續拆到最優;j*(i−j) = 把 i−j 留整塊。因為 dp[i−j] 的定義「一定要拆」不含整塊,所以要另放 j*(i−j)。dp 初始化為 1:當某塊乘上「乘法單位 1」不影響結果,也讓 dp[1]=1 自然成立。

寫法 B · 盡量拆 3(O(n) 數學)// GREEDY 3s

p343_math.cpp C++17 
class Solution {
public:
    int integerBreak(int n) {
        if (n == 2) return 1;                // 2 = 1+1
        if (n == 3) return 2;                // 3 = 1+2

        int product = 1;
        while (n > 4) {                     // 盡量拆出 3
            product *= 3;
            n -= 3;
        }
        return product * n;                    // 剩下 2/3/4 整塊乘進去
    }
};

為什麼拆 3 最優?把一塊 x 拆成 a+b,乘積 a·b 在 a=b 附近最大,但整數裡 3 的「每單位乘積」最高(3^(1/3) ≈ 1.442 > 2^(1/2) ≈ 1.414)。所以盡量湊 3;餘 1 時別留孤單的 1(×1 浪費)→ 拿一個 3 出來變 4=2×2;餘 2 就留一個 2。迴圈到 n≤4 收尾正好處理這些。
