題目簡介// THE QUESTION

QUESTION

把整數 n 拆成 ≥2 個正整數的和,讓這些數的乘積最大,回傳最大乘積。例 10 = 3+3+4 → 3×3×4 = 36。

狀態:dp[i] = 把 i 拆成 ≥2 塊時能得到的最大乘積。你卡住的那行 max(j*dp[i-j], j*(i-j)),正是這題的靈魂 —— 「切」還是「不切」。下面第 2、3 節專門講清楚。

2

切 vs 不切 · 兩項的意義// THE TWO TERMS

核心 · 為什麼要 max 兩項

在切點 j 把 i 分成 j 和 i−j 兩塊。對另一塊 i−j,有兩個選擇:

● 不切 i−j(留整塊)→ 乘積 = j × (i−j)
● 切 i−j(繼續拆到最優)→ 乘積 = j × dp[i−j]

取 max,再對所有 j 取 max,就是 dp[i]。關鍵理由:dp[i−j] 的定義是「一定要再拆」,所以它不包含「把 i−j 留整塊」的情形 —— 那個情形只能靠 j×(i−j) 這一項補上。少了它,就會漏掉「不切更好」的解。

什麼時候「不切」贏、什麼時候「切」贏?(實測)

情形不切 j×(i−j)切 j×dp[i−j]誰贏
i=4, j=22×2 = 42×dp[2]=2×1 = 2不切(小塊別切)
i=6, j=33×3 = 93×dp[3]=3×2 = 6不切
i=8, j=33×5 = 153×dp[5]=3×6 = 18切(大塊繼續拆)

直覺:2、3 這種小塊留整塊最划算(拆了反而變小:dp[2]=1<2、dp[3]=2<3);≥4 的塊才值得繼續拆(dp[5]=6>5)。兩項 max 就是讓 DP 自動在每個 j 上做這個判斷。

3

你的迷思 · dp[2] 是 1 不是 2// MUST-BREAK

你註解寫 dp[2]=2 —— 這正是混淆的根源。看你採用的定義:

dp[i] = 「一定要拆 ≥2 塊」的最大乘積

在這個定義下,dp[2] 只能是 2=1+1 → 乘積 1(不是 2);dp[3]=1×2=2(不是 3)。因為「2 整塊」「3 整塊」不算拆,不在 dp 裡。

正因為 dp[i−j] 把「整塊」排除掉了,轉移才必須另外放 j×(i−j) 來代表「i−j 留整塊」。兩件事是配套的:定義成「一定拆」→ 就得補「不切」項。若你把 dp 定義成「可拆可不拆(允許整塊)」,那 dp[2]=2、轉移就只需 j×dp[i−j] 一項 —— 也對,但那是另一種寫法。兩種定義各自自洽,別混用。

4

核心策略 · 三步驟// STRATEGY

  1. 狀態:dp[i] = 把 i 拆成 ≥2 塊的最大乘積;dp[0]=dp[1]=1(當中性乘法單位)。
  2. 雙層:i 從 2 到 n;切點 j 從 1 到 i−1。
  3. 轉移:dp[i] = max(dp[i], j*(i−j), j*dp[i−j])(不切 / 切,取大);回傳 dp[n]。

複雜度 + 數學捷徑// SPLIT INTO 3s

DP:T = O(n²)　　S = O(n)　　數學:O(n)/O(1)

DP 雙層 n²(n≤58 秒過)。數學捷徑:最優拆法是盡量拆成 3(餘 1 時把一個 3 換成 4=2+2、餘 2 時留一個 2)。因為 3 的「每單位乘積效率」最高(3^(1/3) > 2^(1/2))。

7

一句洞察// INSIGHT

dp[i−j] 逼你「一定再拆」,所以要另放 j×(i−j) 代表「不切、留整塊」。 小塊(2、3)別切、大塊(≥4)才切 —— 兩項 max 讓 DP 自動選。
