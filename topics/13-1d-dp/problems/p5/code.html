寫法 A · 區間 DP(2-D 表)// INTERVAL DP

p5.cpp C++17 
class Solution {
public:
    string longestPalindrome(string s) {
        int n = s.size();
        vector<vector<bool>> dp(n, vector<bool>(n, false));
        pair<int, int> ans = {-1, -1};

        // base ①:長度 1 一定回文
        for (int i = 0; i < n; i++) {
            dp[i][i] = true;
            ans = {i, 1};
        }

        // base ②:長度 2,頭尾相同即回文
        for (int i = 0; i < n - 1; i++) {
            if (s[i] == s[i + 1]) {
                dp[i][i + 1] = true;
                ans = {i, 2};
            }
        }

        // 轉移:長度由小到大,內層 dp[i+1][j-1] 已就緒
        for (int L = 3; L <= n; L++) {
            for (int i = 0; i + L - 1 < n; i++) {
                int j = i + L - 1;
                if (s[i] == s[j] && dp[i + 1][j - 1]) {
                    dp[i][j] = true;
                    ans = {i, L};
                }
            }
        }

        return s.substr(ans.first, ans.second);
    }
};

ans = {起點, 長度}。因為外層按長度遞增掃,越晚更新的 ans 長度越大,結束時自然是最長解。用 substr(起點, 長度) 取出答案。

寫法 B · 中心擴展// EXPAND · O(1) SPACE

p5_expand.cpp C++17 
class Solution {
public:
    string longestPalindrome(string s) {
        int start = 0, best = 1;
        for (int center = 0; center < (int)s.size(); center++) {
            expand(s, center, center, start, best);       // 奇數長度中心
            expand(s, center, center + 1, start, best);   // 偶數長度中心
        }
        return s.substr(start, best);
    }

    void expand(const string& s, int left, int right, int& start, int& best) {
        int n = s.size();
        while (left >= 0 && right < n && s[left] == s[right]) {
            left--; right++;
        }
        int len = right - left - 1;           // while 多退一步,故 -1
        if (len > best) { best = len; start = left + 1; }
    }
};

同樣 O(n²) 時間,但不需要 DP 表 → 空間 O(1)。每個位置當中心往外擴:(center, center) 抓奇數長度、(center, center+1) 抓偶數長度。迴圈停下時 left/right 各多走一步,所以長度是 right − left − 1。

複雜度分析// COMPLEXITY

T = O(n²)　　S = O(n²)(A) / O(1)(B)

A:填 O(n²) 格表,每格 O(1)。B:n 個中心 × 每次最多擴 O(n) = O(n²),只用幾個變數。

4

邊界與陷阱// PITFALLS

陷阱後果正解
按 i,j 順序填表算 dp[i][j] 時內層還沒算外層跑長度 L 由小到大
漏長度 2 的 base偶數回文全判 false單獨處理 dp[i][i+1]
中心擴展只做奇中心漏掉 "bb" 這類偶回文奇 + 偶兩種中心都做
擴展後長度算錯多算或少算兩端len = right − left − 1

註:vector<vector<bool>> 是位元壓縮特化,存取稍慢但省空間;n ≤ 1000 下 O(n²) 完全夠用。
