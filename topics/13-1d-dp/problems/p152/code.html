寫法 A · 同時記 max 與 min(你的版本)// O(1) SPACE

p152.cpp C++17 
class Solution {
public:
    int maxProduct(vector<int>& nums) {
        int n = nums.size();
        int ans = nums[0], maxP = nums[0], minP = nums[0];

        for (int i = 1; i < n; i++) {
            if (nums[i] < 0) swap(maxP, minP);   // 負數翻轉大小 → 先交換

            maxP = max(maxP * nums[i], nums[i]); // 接上去 or 重新開頭
            minP = min(minP * nums[i], nums[i]);

            ans = max(ans, maxP);              // 全程最大的 maxP
        }

        return ans;
    }
};

max(prev×num, num) 的 num 那項代表「從這裡重新開頭」(前面乘積是 0 或負到不划算時)。swap 放在乘之前:先把「翻轉後該當基底的值」擺到 maxP/minP 對應位置,再統一乘 —— 對正數不 swap、對負數 swap,兩種情況同一行處理。

寫法 B · 前綴 / 後綴乘積// PREFIX · SUFFIX

p152_pfx.cpp C++17 
class Solution {
public:
    int maxProduct(vector<int>& nums) {
        int n = nums.size(), ans = nums[0], prefix = 0, suffix = 0;

        for (int i = 0; i < n; i++) {
            prefix = (prefix == 0 ? 1 : prefix) * nums[i];          // 由左往右
            suffix = (suffix == 0 ? 1 : suffix) * nums[n - 1 - i];  // 由右往左
            ans = max(ans, max(prefix, suffix));
        }

        return ans;
    }
};

巧解:最大乘積子陣列一定是「兩個 0 之間某一段」的前綴或後綴。同時累乘前綴(左→右)與後綴(右→左),遇到 0 就重設為 1(等於從下一格重新開始)。負數個數不管奇偶,從某一端數過去必有一個方向湊出最大值,所以掃一次前綴 + 一次後綴就涵蓋。
