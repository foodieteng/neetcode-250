題目簡介// THE QUESTION

QUESTION

一排房子,nums[i] 是第 i 間的現金。相鄰兩間不能同一晚都搶(會觸發警報)。求今晚不驚動警察能搶到的最多金額。

這是 70 爬樓梯 的「最佳化」表親:爬樓梯是「數方法數(相加)」,這題是「取最大值(max)」。核心是對每間房做一個決定 —— 搶它,還是跳過它。

2

為什麼用 DP// WHY DP

A 貪心:每次搶當下最大的一間 O(n)
局部最大不等於全域最大:[2,1,1,2] 貪心搶中間會漏掉頭尾的 2+2=4。
× REJECT · 反例
B 枚舉所有「不相鄰子集」 O(2ⁿ)
每間搶/不搶兩種,窮舉正確但指數爆炸;有大量重疊子問題。
△ REJECT · TLE
ADOPT
C 一維 DP:每間房「搶 or 不搶」取 max O(n) / O(1)
dp[i] = 到第 i 間為止能搶的最多錢。搶 i:接 dp[i−2] 再加 nums[i];不搶 i:沿用 dp[i−1]。取兩者 max。只依賴前兩項 → O(1) 空間。
● ADOPT · O(n)
3

核心策略 · 三步驟// STRATEGY

  1. 定義狀態:dp[i] = 只考慮前 i 間房、且不觸發警報時的最大金額。
  2. 轉移(二選一):dp[i] = max(dp[i−1], dp[i−2] + nums[i]) —— 不搶 vs 搶。
  3. base + 答案:dp[0]=nums[0]、dp[1]=max(nums[0],nums[1]);答案是最後一格。

關鍵推導// WHY dp[i-2]

LEMMA · 搶了 i 就只能接 dp[i−2]

考慮第 i 間房,只有兩種可能,且互斥又窮盡:

① 不搶 i:那前 i 間的最優,就等於前 i−1 間的最優 = dp[i−1]。
② 搶 i:則 i−1 絕對不能搶(相鄰),所以最多只能延續到 i−2,金額 = dp[i−2] + nums[i]。

兩種情況取 max 就是 dp[i]。關鍵是②:一旦搶 i,i−1 被禁掉,合法的接續起點必須退到 i−2 —— 這就是為什麼是 dp[i−2] 而不是 dp[i−1]。

6

複雜度// COMPLEXITY

T = O(n)　　S = O(n) → 可優化 O(1)

單一迴圈掃過每間房,每格常數次比較。只用到前兩項 → 兩個變數 → O(1) 空間。

一句洞察// INSIGHT

爬樓梯把「最後一步」二選一相加;打家劫舍把「這間房」搶/不搶二選一取 max。 同一套一維 DP 骨架,+ 換成 max(…, dp[i−2]+nums[i]) 就從「數方法」變成「求最大」。
