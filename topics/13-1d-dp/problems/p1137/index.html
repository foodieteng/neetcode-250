題目簡介// THE QUESTION

QUESTION

泰波那契數列:T₀=0, T₁=1, T₂=1,之後 Tₙ₊₃ = Tₙ + Tₙ₊₁ + Tₙ₊₂(每項 = 前三項之和)。給 n,回傳 Tₙ。

這是 70 爬樓梯(費氏) 的推廣:費氏每項加前兩項,泰波那契加前三項。 直接用一維 DP 自底向上填表:給定三個 base,之後 t[i] = t[i−1] + t[i−2] + t[i−3] 一路填到 t[n]。

3

核心策略 · 三步驟// STRATEGY

  1. 特判:n==0 → 0;n≤2 → 1(避免 t[2] 存取到不存在的位置)。
  2. 三個 base:t[0]=0, t[1]=1, t[2]=1。
  3. 轉移 + 順序:i 從 3 到 n,t[i]=t[i−1]+t[i−2]+t[i−3];回傳 t[n]。

和費氏的關係// WINDOW SIZE

LEMMA · 依賴窗口從 2 變 3

費氏(70 爬樓梯)每項加前兩項,對應「每次能跨 1 或 2 階」;泰波那契每項加前三項,對應「每次能跨 1、2 或 3 階」。唯一的差別是「往回看幾項」:窗口大小從 2 變 3。轉移式、填表順序、O(1) 滾動的思路完全一樣,只是多留一個變數、多加一項。

因此 base 也要三個(t[0],t[1],t[2]):轉移用到 i−3,得先有前三項才能從 i=3 起遞推。「依賴前 k 項的線性遞迴」都是同一套骨架,k 決定 base 的數量與滾動變數的數量。

6

複雜度// COMPLEXITY

T = O(n)　　S = O(n) → 可優化 O(1)

一個迴圈從 3 填到 n,每格兩次加法。只用到前三項 → 三個變數 → O(1) 空間。n ≤ 37,答案在 int 範圍內。

7

一句洞察// INSIGHT

費氏加前兩項、泰波那契加前三項 —— 差別只在依賴窗口 2 vs 3。 「依賴前 k 項的線性 DP」都同一套:k 個 base、往回加 k 項、k 個變數滾動 O(1)。
