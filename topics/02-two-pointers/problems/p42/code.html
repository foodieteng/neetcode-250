解法 A · 前綴/後綴最高陣列// O(n)/O(n)

p42_arrays.cpp C++17 
class Solution {
public:
    int trap(vector<int>& height) {
        int n = height.size();
        vector<int> l(n, 0), r(n, 0);
        l[0] = height[0]; r[n - 1] = height[n - 1];

        for (int i = 1; i < n; i++)          // leftMax:L→R 取 max
            l[i] = max(l[i - 1], height[i]);

        for (int i = n - 2; i >= 0; i--)       // rightMax:R→L 取 max
            r[i] = max(r[i + 1], height[i]);

        int water = 0;
        for (int i = 0; i < n; i++)
            water += min(l[i], r[i]) - height[i];   // 水位 − 高度

        return water;
    }
};
三趟掃描:算 leftMax[]、算 rightMax[]、逐格加 min(兩者) − height。清楚直觀,但用了兩個長度 n 的陣列 → O(n) 空間。min(l[i], r[i]) − height[i] 恆 ≥ 0(因為 l[i]、r[i] ≥ height[i]),不會扣成負。

解法 B · 對撞雙指針 O(1)(clean)// O(n)/O(1)

p42_twoptr.cpp C++17 
class Solution {
public:
    int trap(vector<int>& height) {
        int l = 0, r = height.size() - 1;
        int leftMax = 0, rightMax = 0, water = 0;

        while (l < r) {
            if (height[l] < height[r]) {          // 左較矮 → 右邊必有更高牆封頂
                leftMax = max(leftMax, height[l]);
                water += leftMax - height[l];    // 用左側 max 結算
                l++;
            } else {                             // 右較矮(或平手)
                rightMax = max(rightMax, height[r]);
                water += rightMax - height[r];  // 用右側 max 結算
                r--;
            }
        }

        return water;
    }
};
你要的 O(1) 版本。不開陣列,只有 l、r、leftMax、rightMax、water。每步比 height[l] 與 height[r],處理較矮那側:因為對側有更高的牆(rightMax ≥ height[r] > height[l]),較矮側的水位由該側 max 就能確定,water += 該側max − 高度,再內移。一趟 O(n)、O(1) 空間 —— 面試首選這版。
