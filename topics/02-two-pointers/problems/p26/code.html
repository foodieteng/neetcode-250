完整實作 · 寫入指針// O(n)/O(1)

p26.cpp C++17 
class Solution {
public:
    int removeDuplicates(vector<int>& nums) {
        int i = 0;                               // 寫入位 / 目前唯一數

        for (int num : nums) {                   // 快指針:逐一掃
            if (i > 0 && num == nums[i - 1]) continue;  // 和上一個保留值相同 → 跳過
            nums[i++] = num;                     // 新值 → 寫到 i,i 前進
        }
        return i;                                // = 唯一元素個數 k
    }
};
for (int num : nums) 的 num 是快指針讀到的值(複本);i 是慢指針(寫入位)。 比對 nums[i-1](上一個保留的唯一值)—— 已排序保證重複相鄰,所以只需比上一個。因為 i ≤ 快指針位置,nums[i++] = num 覆寫的是已讀過的格子,不會蓋掉還沒讀的資料。

變體 · 每個值最多保留 2 個// LC 80

// LC 80:同骨架,把「比上一個」改成「比倒數第 2 個保留值」 int removeDuplicates(vector<int>& nums) { int i = 0; for (int num : nums) { if (i >= 2 && num == nums[i - 2]) continue; // 已放了 2 個相同 → 跳過 nums[i++] = num; } return i; } 保留至多 2 個:只要 num 和「往前數第 2 個保留值」不同就能再放 → 同一寫入指針骨架

這清楚示範「換保留條件」= 換一族題:保留 1 個比 nums[i-1],保留 2 個比 nums[i-2],保留 k 個比 nums[i-k]。 骨架(慢=寫入位、快=掃描、新值才寫)完全不變,只改那一行判斷。移除某值(LC 27)、移動零(LC 283)也是同一招,只是條件變成「不等於 target」「非零」。
