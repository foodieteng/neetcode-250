題目簡介// THE QUESTION

QUESTION

n 個星球、m 條單向傳送門構成有向圖。把星球分成若干「王國」: 同一王國內任兩個星球都能互相到達。求王國數 k,並輸出每個星球屬於哪個王國。

「彼此都能互達」= 有向圖的強連通分量(Strongly Connected Component, SCC)。 標準解 Kosaraju:兩趟 DFS —— 第一趟在原圖依完成時間排序,第二趟在反圖上依序 DFS,每棵樹剛好框住一個 SCC。

DFS 結束一個節點就壓 stack。完成最晚的在頂端 = 位於「源頭」SCC。
◆ 排序

2nd 反圖 DFS · 依 stack 順序,每樹一個 SCC O(V+E)
把邊全部反向,從 stack 頂端逐個彈出當起點 DFS。因為在反圖上,從「源頭 SCC」的點只到得了自己這一組,一次 DFS 剛好框住一個 SCC。共兩趟 O(V+E)。

核心策略// STRATEGY

  1. 建正反兩張圖:g(原圖)與 rg(每條邊反向)。
  2. 第一趟(g):對每個點 DFS;遞迴完所有後代後把自己 push 進 order stack(後序 = 完成時間序)。
  3. 第二趟(rg):反覆從 order 頂端彈出 u;若尚未分派,k++ 並在反圖上 DFS 把整個可達團標成第 k 組。
  4. 輸出:k 與每點的 comp[i]。

為什麼這樣就對// THE INVARIANT

LEMMA · 完成時間 + 反圖

把每個 SCC 縮成一個點,得到的「縮圖(condensation)」一定是 DAG。第一趟 DFS 的完成時間最晚的節點,必定落在這個 DAG 的源頭 SCC(沒有其他 SCC 指進來的那一組)。

第二趟改走反圖:在反圖裡,源頭 SCC 變成「匯點」——從它的點出發,只走得到自己這一組,跨不到別的 SCC。所以一次 DFS 恰好框住一整個 SCC。標記完就把它移除,stack 上的下一個未分派點又是剩餘圖的源頭,如此類推。完成序保證了每次都從「當前源頭」開始。

一句洞察// INSIGHT

SCC = 有向圖裡彼此互達的一團。Kosaraju:原圖排完成序 → 反圖依序 DFS, 每棵樹一個 SCC。核心是「完成最晚的在源頭、反圖上框得住自己」。
