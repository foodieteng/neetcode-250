完整實作 · 雙堆疊懶搬移// 攤還 O(1)

p232.cpp C++17 
class MyQueue {
    stack<int> st1, st2;            // st1 = in(收 push)· st2 = out(供 pop)
public:
    MyQueue() {}

    void push(int x) { st1.push(x); }    // 只進 in,O(1)

    int pop() {
        int v = peek();                 // 先確保 out 有料(必要時倒)
        st2.pop();
        return v;
    }

    int peek() {
        if (st2.empty()) {            // out 空才倒(懶搬移)
            while (!st1.empty()) {
                st2.push(st1.top());  // in → out,順序反轉
                st1.pop();
            }
        }
        return st2.top();             // out 頂端 = 佇列 front
    }

    bool empty() {
        return st1.empty() && st2.empty();  // 兩個都空才算空
    }
};
peek() 是核心:只有 out 空時才把 in 整個倒過去(反轉一次),否則直接取 out 頂端。pop() 借用 peek() 保證 out 有料,再 out.pop()。push 永遠 O(1)。因為倒過去的元素不會再回 in,每個元素一生只搬一次 → 攤還 O(1)。empty 要兩個堆疊都空才算空。

pop 借用 peek · 不必寫兩次倒// DRY

// pop 和 peek 都需要「out 有料」→ 把倒的邏輯只寫在 peek int pop() { int v = peek(); // peek 內部若 out 空會先倒,回傳 out.top() st2.pop(); // 此刻 out 一定非空,直接 pop return v; } // 若 pop 自己也寫一遍「if(out空) 倒」→ 重複邏輯、易改漏一邊 peek 負責「確保 out 有料 + 回傳頂端」;pop = peek + out.pop()

pop 和 peek 都需要「先把 out 準備好」。與其在兩處各寫一次「若 out 空就倒」,不如把倒的邏輯集中在 peek(),pop() 直接 int v = peek(); st2.pop(); —— 呼叫完 peek() 後 out 保證非空,可安全 pop。DRY(不重複)讓程式更短、也不會改漏其中一處。

邊界與陷阱// PITFALLS

陷阱症狀正解
每次 pop 都倒 out 還有料也倒回 in 再倒回 → O(n)/次 只在 out.empty() 時倒
倒了一半就取 沒把 in 倒乾淨 → 順序錯 while (!in.empty()) 全倒
empty 只看一個 in 有料但 out 空 → 誤判為空 in.empty() && out.empty()
push 時就整理 破壞攤還,變 O(n)/push push 只 in.push,懶到 pop 才倒
pop 沒先 peek out 空時 out.pop() 越界(UB) pop 先呼叫 peek 確保有料

細節走讀// WALKTHROUGH

// push1,2,3, pop, push4, pop, pop(頂端在右) push(1,2,3): in=[1,2,3] out=[] pop(): out 空 → 倒 in→out: out=[3,2,1](頂=1); 取 1 → out=[3,2] 回傳 1 push(4): in=[4] out=[3,2] (4 進 in,out 不動) pop(): out 非空 → 直接取頂 2 → out=[3] 回傳 2 pop(): out 非空 → 直接取頂 3 → out=[] 回傳 3 (下一個 pop 才會再把 in=[4] 倒過去) 回傳序 1,2,3,4 —— 完全 FIFO,且只倒了一次(4 尚未搬)
看 push(4) 後那兩個 pop:out 還有 [3,2],所以直接取、不倒 —— 4 靜靜待在 in,等 out 完全空了才會被倒過去。 「新來的 push 不打擾正在被消費的 out」,兩堆疊各司其職、互不干擾,這就是懶搬移能攤還 O(1) 的運作方式:搬移被延後、且每個元素只搬一次。
